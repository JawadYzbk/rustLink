Features

Atlas has a range of features, and more are in development!

Live Map

See a live interactive map, including Hackable Crates, Cargo Ship, Chinook, Explosions, Vending Machines and Team Members.

Server Statistics

See live server statistics such as current player count and how long since last wipe.

Team Chat

Send and receive chat messages with members in your team.

Vending Machine Search

Search for items up for sale in Vending Machines across the map.

Rust+ Pairing

Supports pairing with your Rust servers through the Rust+ in game menu.

and more!

More features are in development! Feel free to send me feature requests.
